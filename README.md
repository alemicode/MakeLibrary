# MakeLibrary

A lightweight Android/Kotlin library that helps you implement an **Instagram-like Back Stack** with a simple and clean API.

## ✨ Features

- 🚀 Easy integration
- 📱 Instagram-like navigation behavior
- 🏗️ Lightweight and production-ready
- 💯 Written entirely in Kotlin
- 📦 Distributed via JitPack
- 🧩 Minimal setup required

---

# Installation

## Step 1. Add the JitPack repository

Add the JitPack repository to your `settings.gradle` or `settings.gradle.kts`:

```kotlin
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        google()
        mavenCentral()
        maven { url = uri("https://jitpack.io") }
    }
}
```

---

## Step 2. Add the dependency

Replace `Tag` with the latest release version.

```kotlin
dependencies {
    implementation("com.github.alemicode:MakeLibrary:Tag")
}
```

For example:

```kotlin
implementation("com.github.alemicode:MakeLibrary:1.0.1")
```
